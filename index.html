How AI detection actually works

01 — Screen Capture

Every frame, Visage captures a region of your screen using mss — a fast, zero-overhead screen grabber. No game files are touched. The game doesn't know anything is reading its pixels.

02 — YOLO Inference

The captured frame is fed into a custom-trained YOLOv8 model. The model was specifically trained on Roblox player character shapes, so it accurately distinguishes real players from the environment even with varied lighting and angles.

03 — GPU Acceleration

On NVIDIA cards, the model is automatically converted to a TensorRT engine on first launch — this happens once and takes about a minute. After that, inference runs at maximum efficiency for your specific GPU, keeping frame rates high.

04 — Aim Assist

Once a target is detected, Visage calculates the offset between your crosshair and the target's head position and moves the mouse using Windows input APIs — the same way any normal mouse moves. Strength and smoothing are fully adjustable.

05 — Zero Footprint

Because Visage reads pixels and moves a mouse — nothing more — it leaves no footprint inside the game process. There are no injected DLLs, no hooked functions, no modified memory. Anti-cheat systems that scan game memory find nothing, because nothing is there.

Installation

Up and running
in under a minute

No Python knowledge needed. No manual installs. The launcher handles everything automatically — on first run it installs all required components, then launches instantly on every run after that.

⚡ On first launch, Visage will install Python and all required packages automatically. If you have an NVIDIA GPU, it will also compile a TensorRT engine for your card — this takes 1–2 minutes and only happens once.
