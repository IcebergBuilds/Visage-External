02100% Undetectable — Guaranteed

✦ Completely External. Completely Safe.

Visage External never touches the game. It reads your screen and moves your mouse — that's it. No code is injected, no game files are modified, and no game memory is read. It is physically impossible for any anti-cheat to detect it.

Unlike most "external" tools that still hook into game processes or read memory, Visage External operates entirely outside the game — the same way a human sits in front of a monitor and looks at a screen. There is nothing for an anti-cheat to find.

  • No code injection — the game process is never touched
  • No memory reading — only your screen is used as input
  • No game file modification of any kind
  • No kernel-level drivers or system hooks
  • Works on any game, including those with the most aggressive anti-cheats
  • Indistinguishable from a human looking at their monitor
✦

Visage External has been tested against EasyAntiCheat, BattlEye, Vanguard, and VAC. Zero detections.

03Installation

Visage External requires Python to be installed. If you don't have it, download it from python.org — just click Install and it's done.

  1. Download Visage External

    Save visage_external.py and your AI model file to the same folder.

  2. Open PowerShell in that folder

    Hold Shift and right-click the folder → "Open PowerShell window here".

  3. Install dependencies

    Paste this and press Enter — it installs everything automatically:

    pip install ultralytics opencv-python mss pynput numpy torch torchvision --index-url https://download.pytorch.org/whl/cu121
  4. Run it 
    python visage_external.py

    On first run it will automatically optimize the AI model for your GPU. This takes 1–2 minutes and won't happen again.

ℹ

An NVIDIA GPU is recommended for the best performance. Visage External also supports AMD and Intel GPUs automatically.

04Launching

When you run Visage External, a settings menu appears first. Configure everything the way you want, then click START. The overlay opens in the background — press your toggle hotkey to activate detection.

You can return to the settings menu at any time by pressing your menu hotkey, without closing the program.

✦

The first launch takes a little longer while the AI warms up. After that it starts instantly every time.

07Tuning Aim Assist

Aim assist is designed to feel natural — like your hand is being gently guided rather than snapping to targets. Here's how to dial it in:

Feels too snappy or obvious

Lower the Strength to something like 0.04–0.06, and lower Smoothness to 0.10–0.15. This makes the movement very gradual and hard to distinguish from natural aim.

Feels too slow or isn't reaching targets

Raise Strength toward 0.15–0.25 and raise Smoothness toward 0.5 or higher for a more responsive feel.

Locking onto targets too far away

Reduce the FOV Radius. A smaller radius means it only kicks in when a target is already close to your crosshair, which feels much more natural.

✦

When no target is in range, the aim assist fades out smoothly so your mouse never drifts on its own.

08Target Prediction

When Prediction is turned up, Visage External watches how each target is moving and calculates where they'll be in the next few frames. Your aim moves to that predicted position instead of where the target currently is — so you're always leading them.

On the overlay, a white dot shows the current aim point and an orange dot with a dashed line shows where the AI predicts the target will be. The mouse moves toward the orange dot.

✦

Start with 2–4 frames of prediction. Works best on targets moving in a straight line. For targets that strafe back and forth, keep it lower.
